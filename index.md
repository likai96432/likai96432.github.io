# Kevin Li
![Kevin Li]({{ '/assets/images/profile.jpg' | relative_url }})

I am a graduate student at McGill University working on software security and agentic AI risks.

My research focuses on understanding, evaluating, and mitigating risks in modern AI and software systems, especially agentic AI systems and security-sensitive software engineering workflows.

## Research Interests

- Agentic AI risk protection
- AI security and governance
- Code summarization under obfuscation
- Android secret leakage detection

## Selected Projects

### Agentic AI Risk Protection Framework

A framework for identifying, scoring, and mapping risks in agentic AI systems. The goal is to help organizations understand which risks are most relevant to their system and trace risks back to root causes and workflow components.

### Android Checked-in Secrets Detection

A study on detecting hard-coded cloud API keys and secrets in Android applications using static analysis and validation techniques.

### Code Summarization under Obfuscation

A project studying how to generate meaningful code summaries when source code identifiers are obfuscated or information-poor.
