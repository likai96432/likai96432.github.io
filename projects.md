# Projects

## Agentic AI Risk Protection Framework

A framework for helping organizations identify, score, and mitigate risks in agentic AI systems.

The framework includes:

- Risk handbook
- System profiling questionnaire
- Risk scoring engine
- Protection mapping
- Root-cause and workflow-component tracing

## Android Secret Leakage Detection

A research project studying hard-coded cloud secrets in Android APKs.

The project compares multiple detection approaches, including:

- Regex and entropy-based filtering
- Static backward dataflow analysis
- Machine-learning-based secret detection

## Code Summarization under Obfuscation

This project studies how to generate useful summaries for obfuscated Java and Android methods.

The current direction explores:

- Method information scoring
- Caller/callee context propagation
- Generated method-name replacement
- LLM-based summary evaluation
