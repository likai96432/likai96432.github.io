# Research

My research interests are broadly in AI security, software engineering, and risk analysis for agentic AI systems.

## Agentic AI Risk Protection

I am working on a framework for analyzing risks in agentic AI systems. The framework connects risk categories, attack chains, system components, and protection methods.

Topics include:

- Prompt injection
- Data exposure
- Cross-tenant isolation failure
- Sandbox escape
- Misconfiguration of access
- Lack of governance
- Resource exhaustion
- Wrong tool selection

## Software Security

I have worked on detecting checked-in secrets in Android applications, especially cloud API keys embedded in APKs.

This work studies:

- Static analysis for secret detection
- Validation of exposed cloud keys
- Limitations of existing detection tools
- Long-tail Android application security

## Code Summarization under Obfuscation

I am also interested in code summarization when source code has lost semantic information due to obfuscation.

This project studies:

- Information loss in method identifiers
- Context-aware code summarization
- Caller/callee-based summary propagation
- Evaluation of generated summaries
